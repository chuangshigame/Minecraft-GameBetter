# Minecraft-GameBetter

##GameBetter是什么?
一个由创世独立完成的运行在1.16.5forge上mod,最新版本约添加了340个元素，包含了大量的武器，工具，方块，维度
##GameBetter有哪些功能？

* 五色的宝石
    *  合成彩色宝石，以进入彩虹维度
    *  更好的工具
* 更好的矿物获取方式
* 更多的游戏唱片
* 星际探索(未来计划)
* 压缩石头工具
* 焕然一新的农场
* 具有挑战性的怪物
* 复杂多变的机器
* .........................
